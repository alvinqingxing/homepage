## 🛠️ Tech Stack

This project is built using vanilla technologies to ensure maximum performance and minimal overhead:

* **HTML5:** Semantic markup structure.
* **CSS3:** Custom styles and layouts (including background video integration).
* **JavaScript:** Vanilla JS for client-side interactions and dynamic functionality.

---

## ✨ Features

* **Vanilla & Lightweight:** Zero frameworks or heavy dependencies used, ensuring blazing-fast load times.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Immersive Experience:** Includes background media elements like `jellyfish.mp4` for enhanced visual appeal.
